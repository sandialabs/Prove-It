allRequirements()[source]¶

Returns the set of axioms that are required (directly or indirectly) by the theorem. Also, if the given theorem is not completely proven, return the set of unproven theorems that are required (directly or indirectly). Returns this axiom set and theorem set as a tuple.

Record the given proof as the proof of this theorem. Updates dependency links (usedAxioms.txt, usedTheorems.txt, and usedBy.txt files) and proven dependency indicators (various provenTheorems.txt files for theorems that depend upon the one being proven) appropriately.
